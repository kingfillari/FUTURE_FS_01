# Fillimon Gebre (KingFillari) – Professional Portfolio

Task 1 for Future Interns Full Stack Web Development internship.

## github

https://github.com/kingfillari/FUTURE_FS_01.git
## Features

- Interactive resume & portfolio sections
- Contact form with email notifications (EmailJS)
- Fully responsive (mobile, tablet, desktop)
- SEO-friendly meta tags
- Smooth scrolling navigation

## Tech Stack

- React 18
- EmailJS (email handling)
- React Icons
- CSS (custom)

## Setup Instructions

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your EmailJS credentials (get from emailjs.com)
4. Run `npm start`
